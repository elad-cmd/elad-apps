# מתודולוגיה ומחקר — בנק שאלות מבחן הידע לרפואה

מסמך זה מרכז את כל הניתוחים, האפיונים והשיטות שפיתחנו, כך שאפשר להמשיך ולייצר עוד תוכן באותו סגנון. עודכן: 8 ביוני 2026.

## 1. הקובץ הקנוני

`questions.js` (בשורש האפליקציה) הוא מקור האמת היחיד — 609 שאלות, נטען ישירות ע"י דפי האתר.

מבנה: `const PRACTICE = { biochem:[...], mol:[...], cell:[...], physio:[...] }`

סכמת שאלה בודדת:
```
{
  "topic": "שם הנושא",
  "q": "גוף השאלה",
  "o": ["אופציה א", "אופציה ב", "אופציה ג", "אופציה ד"],
  "a": 2,            // אינדקס התשובה הנכונה (0-3)
  "e": "ההסבר",
  "new2026": true    // מסומן רק בשאלות שיוצרו ביוני 2026 (לבקרה/מעקב)
}
```

## 2. פורמט ההסבר (e) — תבנית לשימוש חוזר

כל הסבר בנוי כך:
```
<העיקרון / מדוע התשובה נכונה>. ניתוח מסיחים: (אות) <מדוע שגוי>; (אות) <מדוע שגוי>; (אות) <מדוע שגוי>. טיפ: <טיפ קצר לזכירה>.
```

כללי הסגנון:
- האותיות (א/ב/ג/ד) ממופות למיקום האופציה במערך `o`. מנתחים רק את שלושת המסיחים ומדלגים על אות התשובה הנכונה.
- אורך מומלץ: כ-250–400 תווים.
- מונחים לועזיים/טכניים: תעתיק עברי ואז המונח באנגלית בסוגריים בהופעה ראשונה (למשל: ספלייסוזום (spliceosome)). בלי משפטים שלמים באנגלית.
- לגוון את אינדקס התשובה הנכונה (`a`) — לא להשאיר הכול 0.

### תהליך הייצור שעבד הכי טוב
כותבים פריט **מובנה** ביניים, ואסמבלר בונה ממנו את השאלה הסופית — כך מיפוי האותיות בהסבר תמיד נכון אוטומטית:
```
{ "topic", "idx" (0-3), "q", "correct", "d": [{"t","why"} ×3], "principle", "tip" }
```
האסמבלר: ממקם את `correct` במיקום `idx`, ממלא את המסיחים בשאר המיקומים, ובונה את `e` = `principle` + ניתוח המסיחים לפי מיקומם + `tip`.

## 3. ניתוח סילבוס וכיסוי (מיקוד 2026)

ספרי הלימוד הרשמיים:
- ביוכימיה — Lehninger, מהדורה 8
- ביולוגיה מולקולרית + ביולוגיה של התא — Alberts, מהדורה 7
- פיזיולוגיה — Berne & Levy, מהדורה 8

מבנה המבחן: 120 שאלות, 30 לכל מקצוע, ללא הורדת ניקוד על טעות.

**ממצא מרכזי:** סילבוס הפיזיולוגיה ל-2026 מצומצם בכוונה ל-6 פרקים בלבד (תפקוד תא וממברנה, נוזלי גוף, תאי עצב, פוטנציאל פעולה, העברה סינפטית, שריר השלד). לב, כלי דם, נשימה, כליה, אנדוקרינית ושריר חלק **אינם** במיקוד 2026. בבנק יש 25 שאלות פיזיולוגיה שמחוץ למיקוד — הן נשמרות לקראת 2027 (לסמן `focus2026:false` בעת הצורך).

**מצב כיסוי נוכחי:** 609 שאלות. כל נושאי המיקוד מכוסים — מינימום 3 שאלות לנושא, רובם 5+. כלומר 100% מהחומר הנדרש ל-2026. פילוח: ביוכימיה 130 · מולקולרית 224 · תא 141 · פיזיולוגיה 114.
