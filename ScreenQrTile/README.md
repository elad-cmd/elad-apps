# סריקת QR מהמסך — אריח Quick Settings (אנדרואיד)

אפליקציית אנדרואיד נייטיב (Kotlin). מוסיפה אריח להגדרות המהירות.
גורר מלמעלה ← לוחץ "סרוק QR" ← האפליקציה מצלמת פריים בודד של המסך,
מפענחת QR, ואם זה קישור — פותחת אותו מיד. שום דבר לא נשאר רץ ברקע אחרי סריקה.

## בנייה
1. פתח את התיקייה `ScreenQrTile` ב-Android Studio (Giraffe ומעלה).
2. תן לו לסנכרן Gradle (מוריד ZXing + androidx אוטומטית).
3. Run על מכשיר/אמולטור אנדרואיד 8 ומעלה (minSdk 26).

מהטרמינל (אם יש Gradle wrapper): `./gradlew assembleDebug`
ה-APK ייווצר ב-`app/build/outputs/apk/debug/`.

## הוספת האריח
- אנדרואיד 13+: פתח את האפליקציה פעם אחת ולחץ "הוסף את האריח".
- אחרת: גרור הגדרות מהירות ← עריכה (עיפרון) ← גרור את "סרוק QR" לפאנל.

## הזרימה והחיכוך היחיד
צילום מסך דורש אישור MediaProjection. אנדרואיד מציגה דיאלוג "התחל כעת"
לפני הצילום (לפעמים בכל פעם). זה הלחיצה הנוספת היחידה.

## קבצים מרכזיים
- `ScanTileService` — האריח; בלחיצה מסגיר את הפאנל ומפעיל לכידה.
- `CaptureActivity` — אקטיביטי שקוף שמבקש את אישור הצילום.
- `ScreenCaptureService` — Foreground service: מצלם פריים, מפענח, מנקה.
- `QrDecoder` — פענוח ZXID מ-Bitmap + זיהוי קישור/טלפון/מייל לפתיחה ישירה.
- `ResultActivity` — מציג טקסט שאינו קישור, עם העתקה.

## הערות
- minSdk 26, targetSdk 34. אריח = API 24+; סוג foreground service
  mediaProjection נאכף ב-API 34.
- אם הפענוח נכשל על מסך מסוים — בדרך כלל הקוד קטן מדי או חתוך. אפשר
  בעתיד להוסיף ניסיון פענוח על אזורי מסך מוגדלים.
